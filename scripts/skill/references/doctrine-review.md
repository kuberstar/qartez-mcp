# Doctrine: Review Before Edit

Goal: Assess risk and required context before touching code.

## Sequence

1. `qartez_outline`
2. `qartez_impact`
3. `qartez_cochange`
4. `qartez_deps`
5. `qartez_context`
6. `qartez_refs`

## Decision gate

Pause if blast radius is unexpectedly high or boundary violations are likely.
